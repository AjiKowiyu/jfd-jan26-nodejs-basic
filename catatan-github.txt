github adalah sebuah website tempat berbagi project bermacam-macam source code

selain github (microsoft),
ada gitlab , bitbucket (atlassian), 

GIT

fungsi utama github:
1. menyimpan source code secara online/cloud
2. kolaborasi antar developer
3. tools/alat untuk menarik source code ke server
4. versioning > bisa kembali ke versi sebelumnya


status dalam git:
Untracked   => file yg belum terdeteksi oleh sistem git.
Modified    => file yg existing, di modifikasi/ dilakukan perubahan isi file.
Added       => file yg sudah masuk tahap/stage add.


langkah-langkah mengirim source code ke github:
1. add      => keluar dari rumah
2. commit   => kita udah on the way (motor, mobil) dan sampe di lokasi
3. push     => masuk ke kantor


git init => mendaftarkan repo lokal kita sebagai repo yg sudah memakai sistem git
git branch -M main => mengganti cabang utama dengan nama main
git remote add origin https://link-ke-repo-github-kita => menghubungkan repo lokal dengan repo cloud tujuan
git config --global user.email "youremail@provider.com"
git config --global user.name "yourAccountName"
git add .   => menambahkan semua files
git commit -m "pesan commit"
git push -u origin main     => finalisasi pengiriman source code

jika push kedua kali dst cukup ketik perintah git push


syarat kolaborasi:
1. invite developer ke repo kita
2. narik perubahan dari collaborator lain: git pull
